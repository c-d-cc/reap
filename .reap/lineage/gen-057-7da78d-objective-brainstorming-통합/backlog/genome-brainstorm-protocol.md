# Objective Brainstorming 프로토콜 도메인 규칙

Objective 단계에 통합되는 brainstorming 프로세스의 상세 규칙이 genome에 없음.
추가 필요 내용:
- 구조화된 질문 프로토콜 (한 번에 하나, 객관식 우선)
- 2-3개 대안 탐색 + 트레이드오프 규칙
- 섹션별 점진적 디자인 승인 흐름
- Spec 자동 리뷰 루프 (subagent, 최대 3회)
- 스코프 분해 감지 규칙 (FR 10개 초과 또는 독립 서브시스템 감지 시)
- 비주얼 컴패니언 서버 아키텍처 (Node.js 내장 모듈, 파일 기반 통신)
