---
description: "REAP Help — Show help, topics, and current state"
---

Run `reap help` and display the results to the user.

If the user specified a topic (e.g. `/reap.help lifecycle`), pass it as an argument:
```
reap help <topic>
```

If the output contains a `prompt` field, follow the prompt instructions to explain the topic or translate the help text.
