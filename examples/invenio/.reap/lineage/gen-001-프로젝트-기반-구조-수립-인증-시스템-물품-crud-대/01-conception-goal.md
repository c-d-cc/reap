# Generation Goal — Gen-001

## 목표
인벤토리 관리 시스템의 기반을 구축한다: 프로젝트 구조, 인증(회원가입/로그인), 물품 CRUD, 간단한 대시보드.

## 완료 조건
- [ ] 회원가입/로그인 API 동작 (JWT 토큰 발급)
- [ ] 물품 CRUD API 동작 (생성/조회/수정/삭제)
- [ ] 프론트엔드에서 로그인 → 물품 목록 → 추가/편집/삭제 가능
- [ ] 대시보드에서 총 물품 수, 총 재고 가치, 저재고 물품 표시
- [ ] `bun test` 통과
- [ ] `bunx tsc --noEmit` 통과

## 범위
- **관련 Genome 영역**: principles.md (전체), conventions.md (전체), constraints.md (전체)
- **예상 변경 범위**: server/, client/, shared/ 디렉토리 생성 및 구현
- **제외 사항**: 카테고리 관리, 입출고 트랜잭션, 바코드, 리포트 (세대 2 이후)

## 배경
- 첫 세대: 프로젝트 기반 구조부터 기본 CRUD까지
- REAP 워크플로우 실전 검증 목적으로 개발
- ERP 확장을 염두에 둔 도메인 모델 설계
